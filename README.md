# 🐍 terminal-snake-c

> Classic Snake game in your terminal, written in C with ncurses.


## Features

- Smooth keyboard control (arrows)
- Growing snake, increasing score
- Wall collision and self-collision detection
- Game over screen with restart (press 'r')
- Score display

## Build & Run

### Linux / macOS

```bash
sudo apt-get install libncurses-dev   # Ubuntu/Debian
# or brew install ncurses             # macOS
make
./snake_game
